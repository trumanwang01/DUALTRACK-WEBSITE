01 About

我們是誰，
以及我們為何存在

DUAL TRACK ENTERTAINMENT 雙軌娛樂製作，以「雙軌」為核心方法論——同時運行虛擬與現實、嚴肅與荒誕、商業與藝術，在兩條軌道的交會處創造獨特的文化能量。

我們相信，娛樂不只是消遣，而是這個時代最有效的議題載體。透過原創 IP、虛擬偶像、品牌內容與創作者生態，DUAL TRACK 將情感共鳴轉化為可持續的商業模式。

每一個作品都是一次實驗：用荒誕的形式包裹真實的情感，用輕盈的媒介承載沉重的議題。我們不迴避矛盾，而是讓矛盾成為創作的燃料。

在這個注意力碎片化的時代，DUAL TRACK 選擇走向深度。我們不追求最大公約數的流量，而是尋找願意一起走進這個世界的共鳴者。

02 Core Spirit

核心精神

C.01
Creation
創作

創作是我們的根本語言。不是為了填補內容缺口，而是因為有話要說、有世界要建。每一個 IP 都是一個有生命的宇宙，等待被探索、被延伸、被共同書寫。

C.02
Curiosity
好奇

好奇心是我們拒絕固化的理由。對人性的好奇、對形式的好奇、對邊界的好奇——驅動我們不斷問：如果不按牌理出牌，會發生什麼？這種不安分，是我們最珍貴的資產。

C.03
Creativity

創意不是靈光一現，而是一種有紀律的思考方式。我們在限制中尋找自由，在熟悉的形式裡埋下陌生感，讓每一次接觸都留下一點點改變觀點的可能。

04 — Virtual IP

虛擬 IP，
真實的情感連結

虛擬偶像不是替代品，而是一種新的存在形式。在這個邊界模糊的時代，虛擬角色能夠承載現實人物難以觸碰的情感維度——不受形體限制，不受年齡侷限，不受物理距離約束。

DUAL TRACK 的虛擬 IP 戰略不只是創造角色，而是建構一個可以讓粉絲「住進去」的世界觀。每一個細節都是邀請，每一次互動都是共同書寫的故事。
